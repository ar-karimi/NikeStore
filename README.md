# NikeStore
This is a sample online store for checking and buying Nike shoes

Using: MVVM, Koin, Retrofit, RxJava2, Room, LiveData, DataBinding/BindingAdapters, Junit/Espresso, Fresco, EventBus, Material Design, ViewPager2

> Developed in 2023
